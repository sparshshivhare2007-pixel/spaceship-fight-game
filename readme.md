# 🚀 Spaceship Fight Game - Telegram Mini App

A real-time multiplayer spaceship combat game built as a Telegram Mini App with room-based matchmaking.

## ✨ Features

- 🎮 **4 Game Modes**: 1v1, 2v2, Squad (4 players), 4v4 Epic Battles
- 🏠 **Room System**: Create private rooms with unique 8-digit codes
- 👑 **Host Control**: Only room creator can start the game
- 🔫 **Real-time Combat**: Smooth 60 FPS spaceship battles
- 🤖 **Telegram Integration**: Full Telegram Mini App support
- 📊 **Leaderboards**: Track player stats and rankings
- 💾 **Database**: SQLite storage for game history

## 🛠️ Tech Stack

**Frontend:**
- React 18
- Canvas API
- Socket.IO Client
- Telegram WebApp SDK

**Backend:**
- Node.js + Express
- Socket.IO
- SQLite3

**Bot:**
- Telegraf Framework
